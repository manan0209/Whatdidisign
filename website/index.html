Ready to Get Started?

Download WhatDidISign Now

100% Free • No account required • Works immediately after install

1

Download the Extension

Click the button below to download the latest version of WhatDidISign from GitHub Releases.

Download Latest Release
Note: Look for the WhatDidISign-v1.0.0.zip file in the Assets section.
2

Extract the ZIP File

After downloading, extract the ZIP file to a folder on your computer.

Windows: Right click the ZIP file → "Extract All"
Mac: Double click the ZIP file to extract
Linux: Right click → "Extract Here"
Important: Remember where you extracted the files - you'll need this folder location.
3

Open Chrome Extensions Page

Open Google Chrome and navigate to the Extensions management page.

Method 1: Type chrome://extensions/ in the address bar
Method 2: Chrome Menu (⋮) → More Tools → Extensions
4

Enable Developer Mode

In the top right corner of the Extensions page, toggle "Developer mode" ON.

Developer mode
When enabled, you'll see new buttons appear: "Load unpacked", "Pack extension", "Update"
5

Load the Extension

Click the "Load unpacked" button and select the folder you extracted in Step 2.

  1. Click "Load unpacked"
  2. Navigate to your extracted folder
  3. Select the entire folder (not individual files)
  4. Click "Select Folder"
✓

You're All Set!

WhatDidISign should now appear in your extensions list and browser toolbar.

Extension icon visible in toolbar
Orange dots appear on websites with Terms/Privacy links
Click dots to get AI analysis

Troubleshooting

Make sure you selected the correct folder containing the manifest.json file. If it still doesn't work, try refreshing the Extensions page and loading again.

Make sure Developer mode is toggled ON. The toggle should be blue/enabled in the top-right corner of the Extensions page.

Try refreshing the webpage after installing the extension. Some sites may take a moment to scan for legal document links.
